# 工程进度款月报表打印自动化（Excel VBA）

## 背景
每月需对月报表中100+分表上万条数据的格式调整，手动调整耗时约8小时/月，易出错。

## 功能
- 自动探查各分项工程分表结构
- 罗列最终打印目录（可修改）
- 根据需求设置单/双面打印模式
- 一键生成统一格式的打印模板
- 对可能出现的错误信息进行汇报

## 效果
- 处理时间压缩至 **5分钟/月**（压缩99%）
- 准确率100%

## 使用方法
1. 打开 .xlsm 程序，允许宏
2. 点击“开始”按钮
3. 选择目标文件地址及打印模式
4. 输出文件自动保存至程序所在文件夹下
